​Descrição:
- Desenvolvimento de uma plataforma de e-commerce via terminal em C++ com arquitetura modular, inspirada no modelo da OLX. O sistema utiliza orientação a objetos para gerenciar perfis híbridos (comprador e anunciante) e categorias de produtos novos e usados através de persistência de dados em arquivos .txt.

Objetivo:
- Resolver a fragmentação entre os papéis de compra e venda em um único login e facilitar a negociação direta entre usuários. O foco está em permitir a categorização dinâmica de itens e a combinação de preços e pontos de entrega via chat integrado no terminal.  

Motivação para a escolha:
- Este projeto é uma oportunidade prática para dominar o desenvolvimento em C++ ao transformar a lógica da OLX em um sistema via terminal. Através dele, é possível treinar a criação de classes e objetos para modelar usuários e produtos, além de aplicar o encapsulamento ao separar a interface da lógica de gerenciamento. A implementação de persistência de dados em arquivos .txt ensina a salvar e carregar informações com segurança, enquanto a gestão de perfis híbridos e categorias permite explorar hierarquias e relacionamentos entre objetos. Por fim, o sistema de login e a negociação via chat consolidam o domínio sobre a lógica de estados e a interação dinâmica com o usuário.
